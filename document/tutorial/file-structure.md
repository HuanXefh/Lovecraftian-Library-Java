# File Structure

This lib is mostly based off JavaScript, you can find scripts in "assets/scripts", which are categorized into several folders.

| Folder name | Description                                             | Note                   |
|-------------|---------------------------------------------------------|------------------------|
| auxFi       | Various auxiliary files.                                |                        |
| cls         | Lovec class files.                                      |                        |
| ct          | Scripts that create contents using content template.    | Not found in LovecLib. |
| db          | DB object files.                                        |                        |
| frag        | FRAG_xxx module files, similar to MDL_xxx.              |                        |
| glb         | Global parameter files.                                 |                        |
| lib         | External lib files.                                     |                        |
| math        | Module files for mathematical calculation.              |                        |
| mdl         | Common MDL_xxx module files.                            |                        |
| mod         | Module files for cross-mod methods.                     |                        |
| run         | Scripts that should be run only once.                   |                        |
| temp        | Content template files.                                 |                        |
| tempExt     | Content template files for other mods.                  |                        |
| tp          | TP_xxx files for registration of objects like `Effect`. |                        |
